# ShellEcoMarathon

###This website has 5 pages

###Requirements:
python 3.6
```shell
sudo apt-get install python3.6
```

django 2.0
```shell
sudo pip3 install django==2.0
```
To run server:
```shell
cd ShellEcoMarathon-master/ShellEco

python3 manage.py runserver
```

And then go to 127.0.0.1:8000 in your web browser
